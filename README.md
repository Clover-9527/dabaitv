# dabaitv

Java Web课程设计，基于servlet+jsp+ajax+mysql做的影视管理系统

# 运行环境：

Tomcat  9.0	JDK 1.8	MySQL 8.0



后台管理账号密码均为：root，项目依赖：lib 目录
